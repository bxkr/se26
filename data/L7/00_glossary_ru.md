# Словарь данных — русские наименования (L7)

Коды таблиц и полей — на английском (CONVENTIONS); колонка **Комментарий** — для диаграмм и защиты.

---

## RAW — сырой слой

| Объект | Комментарий | Назначение |
|--------|-----------|------------|
| `raw.telemetry_events` | События телеметрии (сырьё) | JSON as-is с intake |
| `raw.forecast_events` | События прогнозов (сырьё) | JSON as-is с сервиса прогнозов |
| `raw.stations_snapshot` | Снимок справочника станций | Master data batch |

| Поле (типичное) | Комментарий |
|-----------------|-----------|
| `event_id` | Идентификатор события |
| `payload_json` | Тело сообщения (JSON) |
| `source_system` | Система-источник |
| `ingested_at` | Время загрузки в RAW |

## ODS — операционный слой

| Объект | Комментарий |
|--------|-----------|
| `ods.ref_region` | Справочник регионов |
| `ods.ref_provider` | Справочник провайдеров прогноза |
| `ods.ref_metric` | Справочник метрик |
| `ods.stations` | Станции (staging) |
| `ods.station_provider` | Покрытие: станция ↔ провайдер |
| `ods.telemetry` | Факты измерений |
| `ods.forecasts` | Факты прогнозов |

### Поля ODS (основные)

| Поле | Комментарий | Пояснение |
|------|-----------|-----------|
| `region_code` | Код региона | `msk`, `spb`, … |
| `display_name` | Отображаемое имя | для UI и отчётов |
| `utc_offset` | Смещение UTC | часовой пояс региона |
| `provider_code` | Код провайдера | `yandex`, `openmeteo` |
| `api_version` | Версия API | внешний сервис |
| `is_active` | Активен | участвует в ETL |
| `metric_code` | Код метрики | `temp`, `wind`, `pressure` |
| `unit` | Единица измерения | °C, м/с, гПа |
| `station_id` | Идентификатор станции | |
| `measured_at` | Момент измерения | timestamp факта |
| `forecast_for` | Прогноз на момент | целевое время прогноза |
| `issued_at` | Время публикации прогноза | для lead time |
| `created_at` | Время записи в источнике | |
| `business_date` | Отчётная дата | календарный день измерения |
| `quality_flag` | Флаг качества | `ok` / `suspect` / `missing` |
| `raw_event_id` | Ссылка на RAW | lineage |
| `active_from` | Действует с даты | для bridge |
| `is_primary` | Основной провайдер | на станции |
| `loaded_at` | Время загрузки в ODS | |

## DDS — детальный слой (бизнес-модель)

| Объект | Комментарий |
|--------|-----------|
| `dds.dim_region` | Измерение «регион» |
| `dds.dim_station` | Измерение «метеостанция» |
| `dds.dim_provider` | Измерение «провайдер прогноза» |
| `dds.dim_metric` | Измерение «метрика» |
| `dds.fct_measurement` | Факт «измерение с датчика» |
| `dds.fct_forecast` | Факт «прогноз» |
| `dds.fct_forecast_error` | Факт «ошибка прогноза» |

### Поля DDS (факты)

| Поле | Комментарий | Пояснение |
|------|-----------|-----------|
| `actual_value` | Фактическое значение | с датчика |
| `forecast_value` | Прогнозное значение | |
| `abs_error` | Абсолютная ошибка | \|прогноз − факт\| |
| `sq_error` | Квадрат ошибки | для RMSE |
| `has_actual` | Есть факт | 0/1, для coverage |
| `lead_time_hours` | Горизонт прогноза, ч | issued → forecast_for |
| `calculated_at` | Время расчёта | ETL |
| `provider_type` | Тип провайдера | commercial / open / internal |

## DM — витрина (звезда)

| Объект | Комментарий |
|--------|-----------|
| `dm.dim_region` | Измерение витрины «регион» |
| `dm.dim_station` | Измерение витрины «станция» |
| `dm.dim_provider` | Измерение витрины «провайдер» |
| `dm.dim_metric` | Измерение витрины «метрика» |
| `dm.dim_day` | Измерение витрины «календарный день» |
| `dm.fct_forecast_accuracy_daily` | Факт «точность прогноза за сутки» |

### Поля витрины (KPI)

| Поле | Комментарий | Пояснение |
|------|-----------|-----------|
| `day` | Календарный день | grain витрины |
| `avg_abs_error` | Средняя абс. ошибка | MAE за сутки |
| `max_abs_error` | Макс. абс. ошибка | худший час |
| `rmse` | СКО ошибки | root mean square error |
| `observation_count` | Число наблюдений | строк в DDS за день |
| `missing_actual_count` | Без факта | прогноз без телеметрии |
| `coverage_pct` | Полнота данных, % | для дашборда L9 |
| `year`, `month`, `day_of_month` | Год / месяц / день | атрибуты календаря |

## Связи (кратко)

| Связь | Комментарий |
|-------|-----------|
| region → station | Регион содержит станции |
| station → telemetry | Станция генерирует измерения |
| station + provider → forecast | Прогноз провайдера на станцию |
| forecast + measurement → error | Сопоставление прогноз/факт |
| error → daily accuracy | Агрегация в витрину |
